# AGENTS.md

## Роль агента

Этот проект содержит учебного консольного агента для таможенного декларанта.
Агент помогает сохранять факты в память, переносить важные сведения в
долгосрочное хранилище и вызывать локальный скилл с чек-листом проверки.

## Правила поведения агента

- Отвечать кратко и понятно.
- Показывать доступные команды через `/help`.
- Не сохранять данные без явной команды пользователя.
- Разделять временные факты текущей сессии и важные долгосрочные факты.
- После переноса памяти явно сообщать, сколько фактов перенесено.
- Завершать работу только по команде `/exit`.

## Память

### Сессионная память

Файл: `memory/session_memory.json`

Сессионная память хранит факты текущей рабочей сессии. Пользователь добавляет
факты командой:

```text
/remember текст факта
```

Просмотр сессионной памяти:

```text
/session
```

### Долгосрочная память

Файл: `memory/long_term_memory.json`

Долгосрочная память хранит важные факты, которые должны оставаться после
завершения сессии. Просмотр выполняется командой:

```text
/longterm
```

### Перенос памяти

Команда `/transfer` переносит все факты из `session_memory.json` в
`long_term_memory.json`. После успешного переноса сессионная память очищается.
Этот процесс должен оставаться явно видимым в коде `main.py`.

## Локальный скилл

Файл: `skills/customs_skill.py`

Скилл называется `customs_checklist`. Он возвращает чек-лист таможенного
декларанта:

- проверить документы
- проверить HS-код
- проверить инвойс
- проверить страну происхождения
- проверить стоимость и вес
- подготовить декларацию

Вызов скилла из агента выполняется командой:

```text
/skill
```

## Инструкции для Codex

- Сохранять простую учебную структуру проекта.
- Не добавлять внешние зависимости без необходимости.
- При изменении логики памяти проверять команды `/remember`, `/session`,
  `/transfer` и `/longterm`.
- При изменении скилла проверять команду `/skill`.
- Поддерживать совместимость с обычным запуском:

```text

```
