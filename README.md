# memory-customs-agent

Учебный консольный агент на Python с двумя видами памяти и локальным скиллом
для таможенного декларанта.

Проект показывает базовую идею агентной памяти:

- факты текущей сессии сохраняются отдельно;
- важные факты можно перенести в долгосрочную память;
- агент может вызывать локальный скилл из отдельного файла.

## Как запустить агента

Требуется Python 3.10 или новее.

```bash
cd memory-customs-agent
python main.py
```

После запуска введите `/help`, чтобы увидеть список команд.

## Команды

```text
/remember <факт>  добавить факт в сессионную память
/session          показать сессионную память
/transfer         перенести сессионную память в долгосрочную
/longterm         показать долгосрочную память
/skill            запустить локальный скилл customs_checklist
/help             показать справку
/exit             выйти из агента
```

## Что такое сессионная память

Сессионная память хранится в файле:

```text
memory/session_memory.json
```

Она предназначена для фактов, которые относятся к текущей рабочей сессии.
Например:

```text
/remember Клиент импортирует партию текстиля из Турции
```

Посмотреть сохраненные факты можно командой:

```text
/session
```

## Что такое долгосрочная память

Долгосрочная память хранится в файле:

```text
memory/long_term_memory.json
```

Она предназначена для важных сведений, которые должны сохраняться надолго.
Посмотреть долгосрочную память можно командой:

```text
/longterm
```

## Как работает перенос данных

Команда `/transfer` читает все факты из `session_memory.json`, добавляет их в
`long_term_memory.json`, а затем очищает `session_memory.json`.

Такой перенос явно реализован в коде `main.py`, чтобы было видно три шага:

1. чтение сессионной памяти;
2. запись в долгосрочную память;
3. очистка сессионной памяти.

## Локальный скилл

Локальный скилл находится в файле:

```text
skills/customs_skill.py
```

Функция `customs_checklist()` возвращает чек-лист таможенного декларанта:

- проверить документы;
- проверить HS-код;
- проверить инвойс;
- проверить страну происхождения;
- проверить стоимость и вес;
- подготовить декларацию.

В агенте скилл вызывается командой:

```text
/skill
```
